# Starter-Project
This [play-clj](https://github.com/oakes/play-clj) game skeleton can be used in a clojure workshop
to program a platformer game. It is build with [leiningen](http://leiningen.org/).

## Install
```bash
lein install
```

## Run the game
```bash
lein run
```

## Run a repl
```bash
lein repl
```

## Run tests
```bash
lein test
```
